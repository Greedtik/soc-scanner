# SOC Scanner v15

Portable Linux security triage scanner.

## Run from GitHub

bash <(curl -s https://raw.githubusercontent.com/YOURUSER/soc_scanner/main/run) --all

## Clone

git clone https://github.com/YOURUSER/soc_scanner.git
cd soc_scanner
sudo bash socscan.sh --all

## Available Flags

--runtime
--network
--ports
--process
--reverse-shell
--docker
--users
--all
