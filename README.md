
# SOC Scanner v17

Portable Linux security / incident response scanner.

## Quick Run from GitHub

bash <(curl -s https://raw.githubusercontent.com/Greedtik/soc-scanner/refs/heads/main/run) --all

## Clone and Run

git clone https://github.com/Greedtik/soc_scanner.git
cd soc_scanner
sudo bash socscan.sh --all
